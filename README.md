# Ev-Adoption-and-Population-Analysis
.Developed an end-to-end data analytics project using Python and Power BI to analyze electric vehicle population data. Performed comprehensive data cleaning, preprocessing, missing value handling, and exploratory data analysis (EDA) using Python libraries such as Pandas, Matplotlib, and Seaborn. Built an interactive Power BI dashboard to visualize EV adoption trends, manufacturer performance, vehicle type distribution, geographic insights, electric range analysis, and CAFV eligibility. Generated business insights to understand market growth patterns and electric vehicle adoption across different regions.

*Built interactive Excel dashboards using Pivot Tables and slicers to analyze 100K+ EV registration records.
*Identified EV adoption trends by city, manufacturer, and vehicle type improving market insight reporting.
*Analyzed battery range performance and CAFV eligibility to support infrastructure planning recommendations.
*Created visual reports highlighting Tesla market dominance and post-2020 EV growth acceleration.
FINAL ANALYSIS:
I used Pivot Tables and Excel dashboards to analyze EV adoption trends by city, brand, and model year. I identified Tesla as the dominant manufacturer, observed significant EV growth after 2020, and analyzed electric range improvements across vehicle generations. I also evaluated CAFV eligibility and regional adoption trends to derive infrastructure and policy recommendations
