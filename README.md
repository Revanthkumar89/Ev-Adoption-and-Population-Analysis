# Ev-Adoption-and-Population-Analysis
.Developed an end-to-end data analytics project using Python and Power BI to analyze electric vehicle population data. Performed comprehensive data cleaning, preprocessing, missing value handling, and exploratory data analysis (EDA) using Python libraries such as Pandas, Matplotlib, and Seaborn. Built an interactive Power BI dashboard to visualize EV adoption trends, manufacturer performance, vehicle type distribution, geographic insights, electric range analysis, and CAFV eligibility. Generated business insights to understand market growth patterns and electric vehicle adoption across different regions.
